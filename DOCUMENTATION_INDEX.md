## üî• Quick Start (Combined TL;DR)

If you just want to get started **right now**:

### 1. Start Backend (1 command)
```bash
cd localbackend && node index.js
```
Copy the IP shown (e.g., `http://192.168.10.226:4000`)

### 2. Update Config (1 edit)
Edit `constants/localBackendConfig.ts` line 12:
```typescript
url: 'http://YOUR_IP_HERE:4000',
```

### 3. Start App (1 command)
```bash
bun expo start
```

### 4. Open on Phone (1 scan)
Open Expo Go app ‚Üí Scan QR code

### 5. Done! üéâ
Sign up and start using the app

**That's literally it.**

For details, see any of the docs above.

## üÜò Still Need Help?

### If Documentation Doesn't Help
1. **Check server logs** - Backend terminal shows all requests
2. **Check app logs** - Expo terminal shows all errors
3. **Test with curl** - Verify backend works: `curl http://IP:4000/api/health`
4. **Clear everything** - Sometimes fresh start helps
5. **Check GitHub issues** - Others may have same problem

### Common Issues Not in Docs
- **Antivirus blocking ports** - Add exception for ports 4000, 19000
- **VPN interference** - Disable VPN while developing
- **Multiple network adapters** - May show wrong IP, check all adapters
- **Phone hotspot** - Won't work, needs WiFi router
